# Setup Instructions
- Clone Repo from GitHub
- Setup Virtual Environment
- **Run Data Pipeline**

# Run Data Pipeline
This repository uses [DVC](https://dvc.org/) to manage data pipelines. To run the most up-to-date pipeline into your local repository, run:
```
dvc repro
```
